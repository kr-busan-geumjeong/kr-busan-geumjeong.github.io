# 금정구 · 부산광역시

> 금정구 지역 정보 랜딩 사이트

## 🌐 랜딩 페이지

### **https://kr-busan-geumjeong.github.io/**

금정구는 부산광역시 최북단에 자리 잡은 자치구로, 금정산 능선과 금샘 전설에서 이름을 따온 유서 깊은 고장입니다. 신라 문무왕 시절 의상대사가 창건한 범어사와 국내 최대 규모의 금정산성을 품어 불교문화와 호국 역사가 짙게 배어 있습니다. 부산대학교를 비롯한 여러 대학이 밀집해 젊음의 거리가 형성된 청년문화 중심지이자, 온천천 물길을 따라 시민이 산책과 여가를 즐기는 생활친화형 도시이기도 합니다. 경부고속도로와 부산종합버스터미널이 위치해 영남권과 부산 도심을 잇는 육로 교통의 관문 역할을 맡고 있습니다. 산과 물, 대학과 사찰이 공존하는 이 지역은 자연과 학문, 전통이 어우러진 독특한 색채를 지닌 곳입니다.
